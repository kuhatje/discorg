There's been some recent efforts to stabilize some of the traits in LanceDB for tables (the lancedb::table::BaseTable trait) and databases (the lancedb::database::Database trait) and there has been a request for a Catalog trait as well. These traits will hopefully make it possible for users whose databases are not simply "a collection of folders in a bucket" but are instead spread out in multiple buckets or completely unique arrangements to advertise their table / database locations in some other way (e.g. an external catalog) and still be able to use the lancedb rust/python/node/java packages in the same way they do today. I've created this channel for discussions on these topics, and what sorts of integrations people hope to see.
